00th-Order Method: Golden Search

If we evaluate ff at only one intermediate point of the interval, we cannot narrow the range within which we know the minimizer is located.

We have to evaluate ff at two intermediate points.

  • We choose the intermediate points in such a way that the reduction in the range is symmetric, i.e., a1−a0=b0−b1=ρ(b0−a0),ρ<12. a_1 - a_0 = b_0 - b_1 = \rho(b_0 - a_0), \qquad \rho < \frac{1}{2}.

  • We then evaluate ff at the intermediate points.

    • If f(a1)<f(b1)f(a_1) < f(b_1) then the minimizer must lie in the range [a0,b1][a_0, b_1].
    • If, on the other hand f(a1)≥f(b1)f(a_1) \geq f(b_1), then the minimizer is located at [a1,b0][a_1, b_0].
  • Starting with the reduced range of uncertainty, we can repeat the process to find two new points: a2a_2 and b2b_2.
  • But this is unnecessary: we know x*∈[a0,b1]x^\ast \in [a_0, b_1]
    • Since a1∈[a0,b1]a_1 \in [a_0, b_1], we can set a1=b2a_1 = b_2.
    • Only one function evaluation of ff at a2a_2 is necessary.
  • How do we find the value of ρ\rho that results only in one new evaluation of ff?

00th-Order Method: Golden Search

Finding ρ\rho

We choose ρ\rho so that: 2345ρ(b1−a0)=b1−b2=1−2ρ\phantom{2345} \rho (b_1 - a_0) = b_1 - b_2 = 1-2\rho.

This has the two solutions: 123ρ1,2=12(3±5)\phantom{123} \rho_{1,2} = \frac{1}{2}(3 \pm \sqrt{5}).

  • Since we require that ρ<12\rho < \frac{1}{2}, we must take ρ=3−52≈0.382.\rho = \frac{3-\sqrt{5}}{2} \approx 0.382.

  • Observe that 1−ρ=5−12≈0.618031-\rho = \frac{\sqrt{5}-1}{2} \approx 0.61803 and ρ1−ρ=1−ρ1\frac{\rho}{1-\rho} = \frac{1-\rho}{1}.

    • Dividing a range in the ratio of ρ\rho to 1−ρ1-\rho has the effect that ratio of the shorter segment to the longer equals the ratio of the longer to the sum of the two.
    • This rule is referred to as the golden section.

  • The uncertainty range reduction is 1−ρ1-\rho at each stage.

    • NN steps of reduction using the golden section method reduces the range by the factor (1−ρ)N(1-\rho)^N.

11st-Order Method: Bisection Method

Assumptions: ff is unimodal and continuously differentiable.

Algorithm

  • Let x0=12(a0+b0)x_0 = \frac{1}{2}(a_0 + b_0), the midpoint of the initial uncertainty interval.
  • Evaluate f′(x0)f'(x_0).
    • If f′(x0)>0f'(x_0) > 0, deduce that the minimizer lies to the left of x0x_0.
      • Reduce the uncertainty interval to [a0,x0][a_0, x_0].
    • If f′(x0)<0f'(x_0) < 0, deduce that the minimizer lies to the right of x0x_0.
      • Reduce the uncertainty interval to [x0,b0][x_0, b_0].
    • If f′(x0)=0f'(x_0) = 0, declare x0x_0 the minimizer and terminate the search.
  • With the new uncertainty interval computed, repeat the process iteratively.
    • Compute the midpoint xkx_k and check the sign of f′(xk)f'(x_k) and reduce the uncertainty to the left or right of xkx_k.
    • Declare xkx_k the minimizer if f′(xk)=0f'(x_k) = 0.

Salient features

  • Instead of using the values of ff, the bisection method uses the values of f′f'.

  • At each iteration, the length of the uncertaintly interval is reduced by a factor of 12\frac{1}{2}.

    • After NN steps, the range is reduced by a factor of (12)N(\frac{1}{2})^N.
    • This factor is smaller than in the golden search or Fibonacci methods.

Example

f(x)=x4−14x3+60x2−70x,x∈[0,2]. f(x) = x^4 - 14x^3 + 60x^2 - 70x, \quad x \in [0, 2].

  • If we want a precision of 0.30.3, then we need N>1−log2(310)N > 1 - \operatorname{log}_2(\frac{3}{10}) iterations, i.e., N=3N = 3.

22nd-Order Method: Newton’s Method

  • Construct a quadratic function qq which at xkx_k agrees with ff up to second derivatives, that is

q(x)=f(xk)+f′(xk)(x−xk)+12f″(xk)(x−xk)2. q(x) = f(x_k) + f'(x_k)(x-x_k) + \frac{1}{2}f''(x_k)(x-x_k)^2.

  • Calculate an estimate xk+1x_{k+1} of the minimum point of ff by finding the point where the derivative of qq vanishes.

0=q′(xk+1)=f′(xk)+f″(xk)(xk+1−xk)⇒xk+1=xk−f′(xk)f″(xk).(1) \begin{align} 0 &= q'(x_{k+1}) = f'(x_k) + f''(x_k)(x_{k+1}-x_k) \\ &\Rightarrow x_{k+1} = x_k - \frac{f'(x_k)}{f''(x_k)}. \end{align} \qquad(1)

  • xk+1x_{k+1} resulting from Newton’s method does not depend on the value f(xk)f(x_k).
  • Newton’s method may be viewed as a technique for iteratively solving equations of the form

g(x)=0, g(x) = 0,

where, when applied to minimization, we put g(x)=f′(x)g(x) = f'(x).

Proposition

Let x*x^\ast satisfy g(x*)=0g(x^\ast) = 0, g′(x*)≠0g'(x^\ast) \neq 0. Then, provided x0x_0 is sufficiently close to x*x^\ast, the sequence {xk}k=0∞\{x_k\}_{k=0}^\infty generated by Newton’s method Equation 1 converges to x*x^\ast with an order of convergence at least two.

  • We want to find the minimizer of

  • We want an accuracy of ε=10−5\varepsilon = 10^{-5}, i.e., stop when

|xk+1−xk|<ε. |x_{k+1} - x_k | < \varepsilon.

  • We compute

f′(x)=x−cosx,f″(x)=1+sinx. f'(x) = x - \cos{x}, \quad f''(x) = 1 + \sin{x}.



x1=12−12−cos121+sin12=0.7552,x2=x1−f′(x1)f″(x1)=x1−0.027101.685=0.7391,x3=x2−f′(x2)f″(x2)=x2−9.461×10−51.673=0.7390,x4=x3−f′(x3)f″(x3)=x3−1.17×10−91.673=0.7390. \begin{align} x_1 &= \frac{1}{2} - \frac{\frac{1}{2} - \cos{\frac{1}{2}}}{1 + \sin{\frac{1}{2}}} = 0.7552, \\ x_2 &= x_1 - \frac{f'(x_1)}{f''(x_1)} = x_1 - \frac{0.02710}{1.685} = 0.7391, \\ x_3 &= x_2 - \frac{f'(x_2)}{f''(x_2)} = x_2 - \frac{9.461 \times 10^{-5}}{1.673} = 0.7390, \\ x_4 &= x_3 - \frac{f'(x_3)}{f''(x_3)} = x_3 - \frac{1.17 \times 10^{-9}}{1.673} = 0.7390. \end{align}

Inaccurate Line Search

  • In practice, we do not bend over backwards to find the exact minimum when performing line search.
    • It is often desirable to sacrifice accuracy in the line search routine in order to conserve overall computation time.
    • Most functions to not attain their minimum along the line we’re searching at a particular iteration anyway!
  • Inaccuracy is introduced in a line search algorithm by simply terminating the search before it has converged.

Armijo’s Rule

  • A practical and popular criterion for terminating a line search is Armijo’s rule.

  • The idea is that the rule should first guarantee that the selected α\alpha is not too large, and next it should not be too small. ϕ(α)=f(𝐱k+α𝐝k). \phi(\alpha) = f(\bm{x}_k + \alpha \bm{d}_k).

  • Consider the function ϕ(0)+εϕ′(0)α\phi(0) + \varepsilon \phi'(0)\alpha, for fixed ε\varepsilon, 0<ε<10 < \varepsilon < 1.

  • A value of α\alpha is consered to be not too large if the corresponding function value lies below the dashed line; that is, if ϕ(α)≤ϕ(0)+εϕ′(0)α.(2) \phi(\alpha) \leq \phi(0) + \varepsilon \phi'(0)\alpha. \qquad(2)

  • To ensure that α\alpha is not too small, a value η>1\eta > 1 is selected, and α\alpha is then considered to be not too small if ϕ(ηα)>ϕ(0)+εϕ′(0)ηα. \phi(\eta \alpha) > \phi(0) + \varepsilon \phi'(0)\eta \alpha.

  • This means that if α\alpha is increased by a factor α\alpha, it will fail to meet the test Equation 2.

The acceptable region defined by the Armijo rule when η=2\eta = 2.

The Method of Steepest Descent: First-Order

Often the first tried method on a new problem.

The Method and Convergence

  • The gradient ∇f(𝐱)\nabla f(\bm{x}) is defined as a nn-dim. row vector.
  • We define the nn-dim. column vector g(𝐱)=∇f(𝐱)⊤g(\bm{x}) = \nabla f(\bm{x})^\top.

The Method of Steepest Descent (SDM)

The iterative algorithm is

𝐱k+1=𝐱k−αk𝐠k, \bm{x}_{k+1} = \bm{x}_k - \alpha_k \bm{g}_k,

where the stepsize αk\alpha_k is a nonnegative scalar possible minimizing f(𝐱k−α𝐠k)f(\bm{x}_k - \alpha \bm{g}_k).

The Algorithm

Define the mapping 1𝐒:ℝ2n→ℝn\phantom{1} \bm{S}: \mathbb{R}^{2n} \rightarrow \mathbb{R}^n by 𝐒(𝐱,𝐝)={𝐲:𝐲=𝐱+α𝐝,α≥0,f(𝐲)=min0≤α<∞f(𝐱+α𝐝)}. \bm{S}(\bm{x}, \bm{d}) = \{\bm{y}: \bm{y} = \bm{x} + \alpha \bm{d}, \;\; \alpha \geq 0, \; f(\bm{y}) = \operatorname{min}_{0 \leq \alpha < \infty} f(\bm{x} + \alpha \bm{d}) \}.

This is a closed map if 𝐝≠𝟎\bm{d} \neq \bm{0}.

The overall algorithm is 𝐀:ℝn→ℝn\; \bm{A}: \mathbb{R}^n \rightarrow \mathbb{R}^n which gives 𝐱k+1∈𝐀(𝐱k)\bm{x}_{k+1} \in \bm{A}(\bm{x}_k) can be decomposed in the form

𝐀=𝐒𝐆,𝐆(𝐱)=(𝐱,−𝐠(𝐱)). \bm{A} = \bm{SG}, \quad \bm{G}(\bm{x}) = (\bm{x}, -\bm{g}(\bm{x})).

Global Convergence

  • Define the solution set Γ={𝐱∈ℝn:∇f(𝐱)=𝟎}\Gamma = \{\bm{x} \in \mathbb{R}^n: \nabla f(\bm{x}) = \bm{0}\}.

  • Z(𝐱)=f(𝐱)Z(\bm{x}) = f(\bm{x}) is a descent function for 𝐀\bm{A}, since for ∇f(𝐱)≠𝟎\nabla f(\bm{x}) \neq \bm{0} min0≤α<∞f(𝐱−α𝐠(𝐱))<f(𝐱). \operatorname{min}_{0 \leq \alpha < \infty} f(\bm{x} - \alpha \bm{g}(\bm{x})) < f(\bm{x}).

  • Thus by the Global Convergence Theorem, if the sequence {𝐱k}\{\bm{x}_k\} is bounded, it will have limit points and each of these is a solution.

Definition (First-order β\beta-Lipschitz Function)

For any two points 𝐱\bm{x} and 𝐲\bm{y}, |∇f(𝐲)−∇f(𝐱)|≤β|𝐲−𝐱||\nabla f(\bm{y}) - \nabla f(\bm{x})| \leq \beta |\bm{y} - \bm{x}|, for a positive real number β\beta.

Convergence Speed for fixed stepsize αk=1β\alpha_k = \frac{1}{\beta} is arithmetic.

We may not know 1β\frac{1}{\beta} so we employ a backtracking line search: start with a guess of β\beta; if sufficient obj. reduction is achieved, halve β\beta; otherwise double β\beta. Stop when the process is reversed.

Steepest Descent Analysis

Lemma. Let f(𝐱)f(\bm{x}) be differentiable everywhere and satisfy the (first-order) β\beta-Lipschitz condition. Then, for any two points 𝐱\bm{x} and 𝐲\bm{y} f(𝐲)−f(𝐱)−∇f(𝐱)(𝐲−𝐱)≤β2|𝐲−𝐱|2. f(\bm{y}) - f(\bm{x}) - \nabla f(\bm{x})(\bm{y} - \bm{x}) \leq \frac{\beta}{2}|\bm{y} - \bm{x}|^2.

The Quadratic Case

When f(𝐱)f(\bm{x}) is strongly convex, the convergence speed can be increased from arithmetic to geometric or linear convergence.

We focus on the quadratic problem

Nonquadratic Case

Theorem (Newton’s Method)

Modifications

Although Newton’s method is very attractive in terms of its convergence properties near the solution, it requires modification before it can be used at points that are remote from the solution.

Damping

A search parameter α\alpha is introduced 𝐱k+1=𝐱k−αk𝐅(𝐱k)−1∇f(𝐱k)⊤, \bm{x}_{k+1} = \bm{x}_k - \alpha_k \bm{F}(\bm{x}_k)^{-1}\nabla f(\bm{x}_k)^\top, where αk\alpha_k is selected to minimize ff.

Positive Definiteness and Scaling

General class of algorithms is given by 𝐱k+1=𝐱k+α𝐝k=𝐱k−α𝐌k𝐠k,(14) \bm{x}_{k+1} = \bm{x}_k + \alpha \bm{d}_k = \bm{x}_k - \alpha \bm{M}_k \bm{g}_k, \qquad(14)

  • SD: 𝐌k=𝐈\bm{M}_k = \bm{I}, Newton: 𝐌k=𝐅(𝐱k)−1\bm{M}_k = \bm{F}(\bm{x}_k)^{-1}.

For small α\alpha, it can be shown that f(𝐱k+1)=f(𝐱k)−α𝐠k⊤𝐌k𝐠k+O(α2). f(\bm{x}_{k+1}) = f(\bm{x}_k) - \alpha \bm{g}_k^\top \bm{M}_k \bm{g}_k + O(\alpha^2).

  • As α→0\alpha \rightarrow 0, the second term on the rhs dominates the third.
  • To guarantee a descrese in ff, we must have 𝐠k⊤𝐌k𝐠k>0\bm{g}_k^\top \bm{M}_k \bm{g}_k > 0.
    • Simplest way to ensure this is to require 𝐌k≻𝟎\bm{M}_k \succ \bm{0}.

General Problems

  • In practice, Newton’s method must be modified to accommodate the possible nonpositive definiteness at regions remote from the solution.

  • Common approach: 𝐌k=[μk𝐈+𝐅(𝐱k)]−1\bm{M}_k = [\mu_k\bm{I} + \bm{F}(\bm{x}_k)]^{-1} for some μk>0\mu_k > 0.

  • This can be regarded as a compromise between SD (μk\mu_k very large) and Newton’s method (μk=0\mu_k = 0).

Levenberg-Marquardt performs Cholesky factorization for a given value of μk\mu_k as follows μk𝐈+𝐅(𝐱k)=𝐆𝐆⊤.\mu_k \bm{I} + \bm{F}(\bm{x}_k) = \bm{G}\bm{G}^\top.

This checks for positive definiteness (not positive definite if factorization fails).

If the factorization breaks down μk\mu_k is increased.

Step direction is found by solving 𝐆𝐆⊤𝐝k=−𝐠k\bm{G}\bm{G}^\top \bm{d}_k = -\bm{g}_k.

Coordinate and Stochastic Gradient Descent

The algorithms in this section are sometimes attractive because of their easy implementation. Generally, however, their convergence properties are poorer than SD.

Coordinate Descent

Given a point 𝐱=(x1,x2,…,xn)\bm{x} = (x_1, x_2, \ldots, x_n), descent w.r.t. the coordinate xix_i (ii fixed) means that one solves

minimizexif(x1,x2,…,xn). \operatorname{minimize}_{x_i} \quad f(x_1, x_2, \ldots, x_n).

  • Only changes in the single component xix_i are allowed in seeking a new and better vector 𝐱\bm{x}.
    • One can also consider 𝐱i\bm{x}_i the ithi^{\text{th}} block of variables (block coordinate method).
  • In our general terminology, each such descent can be regarded as a descent in the direction 𝐞i\bm{e}_i.

Cyclic methods

  • minx1,x2,…,xn\operatorname{min}\;\; x_1, x_2, \ldots, x_n \quad then minxn−1,xn−2,…,x1\quad \operatorname{min}\;\; x_{n-1}, x_{n-2}, \ldots, x_1
  • minx1,x2,…,xn\operatorname{min}\;\; x_1, x_2, \ldots, x_n \quad then minx1,x2,…,xn\quad \operatorname{min}\;\; x_{1}, x_{2}, \ldots, x_n

They have the advantage of not requiring any information about ∇f\nabla f to determine descent directions.

Gauss-Southwell Method

  • If the gradient of ff is available, then it is possible to select the order of descent coordinates on the basis of the gradient.

  • At each stage, the coordinate corresponding to the largest component of the gradient vector is selected for descent.

Stochastic Gradient Descent (SGD) Method

Imagine we are solving a stochastic optimization problem or its simple average approximation

f(𝐱)=𝔼[ϕ(𝐱,𝛏)]orf(𝐱)=1M∑i=1Mϕ(𝐱,𝛏𝐢), f(\bm{x}) = \mathbb{E}[\phi(\bm{x}, \bm{\xi})] \qquad \text{or} \qquad f(\bm{x}) = \frac{1}{M} \sum_{i=1}^M \phi(\bm{x}, \bm{\xi_i}),

where 𝛏\bm{\xi} is a random parameter and ξi\xi_i is a randomly chosen sample.

  • If we simply apply the SD, the evaluation of the gradient vector would be costly, involving a large sum computation.
  • The SGD would, at the current iterate 𝐱k\bm{x}_k, randomly select a sample point ξk\xi_k and compute its (sub)gradient vector 𝐠k:=𝐠(𝐱k,ξk)\bm{g}_k := \bm{g}(\bm{x}_k, \xi_k), which satisfies, in expectation, 𝔼[𝐠k|𝐱k]∈∂f(𝐱k)\mathbb{E}[\bm{g}_k | \bm{x}_k] \in \partial f(\bm{x}_k).
  • Then the method would update, starting from an initial solution 𝐱0\bm{x}_0,

𝐱k+1=𝐱k−αk𝐠k, \bm{x}_{k+1} = \bm{x}_k - \alpha_k \bm{g}_k,

until k=K−1k = K-1 and return the average solution: 𝐱‾=1K∑k=0K−1𝐱k\bar{\bm{x}} = \frac{1}{K}\sum_{k=0}^{K-1} \bm{x}_k.
