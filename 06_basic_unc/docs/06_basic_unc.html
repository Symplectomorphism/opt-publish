First-Order Necessary Conditions

Theorem (Weierstrass).

A continuous function ff defined on a compact set SS has a minimum point in SS.

Feasible and Descent Directions

Definition (relative minimum or local minimum).

A point 𝐱*∈Ω\bm{x}^\ast \in \Omega is said to be a relative minimum point of ff over Ω\Omega if ∃ε>0\exists \varepsilon > 0 such that f(𝐱)≥f(𝐱*)f(\bm{x}) \geq f(\bm{x}^\ast) for all 𝐱∈Ω\bm{x} \in \Omega within a distance ε\varepsilon of 𝐱*\bm{x}^\ast.

Definition (global minimum).

A point 𝐱*∈Ω\bm{x}^\ast \in \Omega is said to be a global minimum point of ff over Ω\Omega if f(𝐱)≥f(𝐱*)f(\bm{x}) \geq f(\bm{x}^\ast) for all 𝐱∈Ω\bm{x} \in \Omega.

  • Usually impossible to find w/ gradient-based methods.
  • Along any given direction, the objective function can be regarded as a function of a single variable: the parameter defining movement in this direction.

Feasible direction

Given 𝐱∈Ω\bm{x} \in \Omega we say that a vector 𝐝\bm{d} is a feasible direction at 𝐱\bm{x} if there is an α‾>0\bar{\alpha} > 0 such that 𝐱+α𝐝∈Ω\bm{x} + \alpha \bm{d} \in \Omega for all α\alpha with 0≤α≤α‾0 \leq \alpha \leq \bar{\alpha}.

Descent direction

An element of the set of directions with the property {𝐝:∇f(𝐱)𝐝<0}\{\bm{d}: \nabla f(\bm{x}) \bm{d} < 0\} is called a descent direction.

If f(𝐱)∈C1f(\bm{x}) \in C^1, then there is α‾>0\bar{\alpha} > 0 such that f(𝐱+α𝐝)<f(𝐱)f(\bm{x} + \alpha \bm{d}) < f(\bm{x}) for all α\alpha with 0<α≤α‾0 < \alpha \leq \bar{\alpha}. The direction 𝐝⊤=−∇f(𝐱)\bm{d}^\top = -\nabla f(\bm{x}) is the steepest descent one.

First-Order Necessary Conditions

Proposition (FONC).

If 𝐱*\bm{x}^\ast is a relative minimum point of ff over Ω⊆ℝn\Omega \subseteq \mathbb{R}^n, then for any 𝐝∈ℝn\bm{d} \in \mathbb{R}^n that is a feasible direction at 𝐱*\bm{x}^\ast, we have ∇f(𝐱*)𝐝≥0\nabla f(\bm{x}^\ast) \bm{d} \geq 0.

Proof.

For any α\alpha, 0≤α≤α‾0 \leq \alpha \leq \bar{\alpha}, the point 𝐱(α)=𝐱*+α𝐝∈Ω\bm{x}(\alpha) = \bm{x}^\ast + \alpha \bm{d} \in \Omega. For 0≤α≤α‾0 \leq \alpha \leq \bar{\alpha} define the function g(α)=f(𝐱(α))g(\alpha) = f(\bm{x}(\alpha)). Then gg has a relative minimum at α=0\alpha = 0. By ordinary calculus, we have

g(α)−g(0)=g′(0)α+o(α)(1) g(\alpha) - g(0) = g'(0)\alpha + o(\alpha) \qquad(1)

where o(α)o(\alpha) denotes terms that go to zero faster than α\alpha. If g′(0)<0g'(0) < 0, then for sufficiently small values of α\alpha, the rhs of Equation 1 will be negative and hence g(α)−g(0)<0g(\alpha) - g(0) < 0, which contradicts the minimality of g(0)g(0). Thus g′(0)=∇f(𝐱*)𝐝≥0g'(0) = \nabla f(\bm{x}^\ast) \bm{d} \geq 0.

Corollary (Unconstrained Case).

Let Ω⊆ℝn\Omega \subseteq \mathbb{R}^n and let f∈C1f \in C^1 on Ω\Omega. If 𝐱*\bm{x}^\ast is a relative minimum point of ff over Ω\Omega and if 𝐱*∈Ω̊\bm{x}^\ast \in \mathring{\Omega}, then ∇f(𝐱*)=𝟎\nabla f(\bm{x}^\ast) = \bm{0}.

First-Order Sufficient Conditions

Proposition (FOSC).

Let f∈C1f \in C^1 be a convex function on ℝn\mathbb{R}^n. If 𝐱*\bm{x}^\ast meets the first-order conditions ∇f(𝐱*)=𝟎\nabla f(\bm{x}^\ast) = \bm{0}, 𝐱*\bm{x}^\ast is a global minimizer of ff.

Examples

Example 1. Consider the problem minimizef(x1,x2)=x12−x1x2+x22−3x2. \operatorname{minimize} f(x_1, x_2) = x_1^2 - x_1x_2 + x_2^2 - 3x_2. There are no constraints, Ω=ℝ2\Omega = \mathbb{R}^2. Setting the partial derivatives of ff equal to zero yields

2x1−x2=0,−x1+2x2=3. 2x_1 - x_ 2= 0, \quad -x_1 + 2x_2 = 3.

which has the unique solution x1=1x_1 = 1, x2=2x_2 = 2. This is a global minimum point of ff.

Example 2. minimizef(x1,x2)=x12−x1+x2+x1x2,subject tox1,x2≥0. \begin{align} \operatorname{minimize} & f(x_1, x_2) = x_1^2 - x_1 + x_2 + x_1x_2, \\ \text{subject to} & x_1, x_2 \geq 0. \end{align} This problem has a global minimum at x1=12x_1 = \frac{1}{2}, x2=0x_2 = 0. At this point

∂f∂x1=2x1−1+x2=0,∂f∂x2=1+x1=32. \frac{\partial f}{\partial x_1} = 2x_1 - 1 + x_2 = 0, \quad \frac{\partial f}{\partial x_2} = 1 + x_1 = \frac{3}{2}.

  • Thus the partial derivatives do not both vanish at the solution
  • Since any feasible direction must have an x2x_2 component greater than or equal to zero, we have ∇f(𝐱*)𝐝≥0,∀𝐝∈ℝ2. \nabla f(\bm{x}^\ast) \bm{d} \geq 0, \;\; \forall \bm{d} \in \mathbb{R}^2.

Example 1 – Logistic Regression

  • We have vectors 𝐚i∈ℝd\bm{a}_i \in \mathbb{R}^d for i=1,2,…,n1i = 1, 2, \ldots, n_1 in a class and vectors 𝐛j∈ℝd\bm{b}_j \in \mathbb{R}^d for j=1,2,…,n2j = 1, 2, \ldots, n_2 not in that class.
  • We wish to classify them, i.e., find a vector 𝐲∈ℝd\bm{y} \in \mathbb{R}^d and a number β\beta such that

exp(𝐚i⊤𝐲+β)1+exp(𝐚i⊤𝐲+β)≈1,∀i, and exp(𝐛j⊤𝐲+β)1+exp(𝐛j⊤𝐲+β)≈0,∀j. \frac{\operatorname{exp}(\bm{a}_i^\top \bm{y} + \beta)}{1 + \operatorname{exp}(\bm{a}_i^\top \bm{y} + \beta)} \approx 1, \;\; \forall i, \quad \text{ and } \quad \frac{\operatorname{exp}(\bm{b}_j^\top \bm{y} + \beta)}{1 + \operatorname{exp}(\bm{b}_j^\top \bm{y} + \beta)} \approx 0, \;\; \forall j.

  • This problem can be cast as an unconstrained optimization problem

maximize𝐲,β(∏iexp(𝐚i⊤𝐲+β)1+exp(𝐚i⊤𝐲+β))(∏j(1−exp(𝐛j⊤𝐲+β)1+exp(𝐛j⊤𝐲+β))) \operatorname{maximize}_{\bm{y}, \beta} \left(\prod_i \frac{\operatorname{exp}(\bm{a}_i^\top \bm{y} + \beta)}{1 + \operatorname{exp}(\bm{a}_i^\top \bm{y} + \beta)}\right) \left(\prod_j \left(1 - \frac{\operatorname{exp}(\bm{b}_j^\top \bm{y} + \beta)}{1 + \operatorname{exp}(\bm{b}_j^\top \bm{y} + \beta)} \right) \right)

which may equivalently be expressed using a log transformation as

minimize𝐲,β∑ilog(1+exp(−𝐚i⊤𝐲−β))+∑jlog(1+exp(𝐛i⊤𝐲+β)). \operatorname{minimize}_{\bm{y}, \beta} \sum_i \operatorname{log}\left(1 + \operatorname{exp}(-\bm{a}_i^\top \bm{y} - \beta) \right) + \sum_j \operatorname{log}\left(1 + \operatorname{exp}(\bm{b}_i^\top \bm{y} + \beta) \right).

Second-Order Necessary Conditions

Proposition (Second-Order Necessary Conditions)

Let Ω⊆ℝn\Omega \subseteq \mathbb{R}^n and let f∈C2f \in C^2 on Ω\Omega. If 𝐱*\bm{x}^\ast is a relative minimum point of ff over Ω\Omega, then for any 𝐝∈ℝn\bm{d} \in \mathbb{R}^n that is a feasible direction at 𝐱*\bm{x}^\ast we have

∇f(𝐱*)𝐝≥0(2) \nabla f(\bm{x}^\ast) \bm{d} \geq 0 \qquad(2) ∇f(𝐱*)𝐝=0, then 𝐝⊤∇2f(𝐱*)𝐝=𝐝⊤𝐅(𝐱*)𝐝≥0.(3) \nabla f(\bm{x}^\ast) \bm{d} = 0, \;\; \text{ then } \;\; \bm{d}^\top \nabla^2 f(\bm{x}^\ast) \bm{d} = \bm{d}^\top \bm{F}(\bm{x}^\ast) \bm{d} \geq 0. \qquad(3)

Proof

The first condition is just the FONC, and the second applies only if ∇f(𝐱*)𝐝=0\nabla f(\bm{x}^\ast)\bm{d} = 0. In this case, introducing 𝐱(α)=𝐱*+α𝐝\bm{x}(\alpha) = \bm{x}^\ast + \alpha \bm{d} and g(α)=f(𝐱(α))g(\alpha) = f(\bm{x}(\alpha)) as before, we have, in view of g′(α)=0g'(\alpha) = 0,

g(α)−g(0)=12g″(0)α2+o(α2). g(\alpha) - g(0) = \frac{1}{2}g''(0)\alpha^2 + o(\alpha^2).

If g″(0)<0g''(0) < 0a the rhs of the above equation is negative for sufficiently small α\alpha which contradicts the relative minimum nature of g(0)g(0). Thus

g″(0)=𝐝⊤∇2f(𝐱*)𝐝≥0. g''(0) = \bm{d}^\top \nabla^2 f(\bm{x}^\ast)\bm{d} \geq 0.

  • See Example 2 from FOSC slide with d2=0d_2 = 0 whence 𝐝⊤∇2f(𝐱*)𝐝=2d12≥0\bm{d}^\top \nabla^2 f(\bm{x}^\ast) \bm{d} = 2d_1^2 \geq 0, satisfying the second condition.

Second-Order Necessary Conditions

Proposition (Second-Order Necessary Conditions — Unconstrained Case)

Let 𝐱*∈Ω̊\bm{x}^\ast \in \mathring{\Omega} and suppose 𝐱*\bm{x}^\ast is a relative minimum point over Ω\Omega of f∈C2f \in C^2. Then

∇f(𝐱*)=0(4) \nabla f(\bm{x}^\ast) = 0 \qquad(4) ∀𝐝,𝐝⊤𝐅(𝐱*)𝐝≥0.(5) \forall \bm{d}, \;\; \bm{d}^\top \bm{F}(\bm{x}^\ast) \bm{d} \geq 0. \qquad(5)

  • The second condition is equivalent to stating that the matrix 𝐅(𝐱*)\bm{F}(\bm{x}^\ast) is positive semidefinite.

Example

minimizef(x1,x2)=x13−x12x2+2x22subject tox1,x2≥0. \begin{align} \operatorname{minimize} & f(x_1, x_2) = x_1^3 - x_1^2x_2 + 2x_2^2 \\ \text{subject to} & x_1, x_2 \geq 0. \end{align}

If x1,x2>0x_1, x_2 > 0, then the FONC are

3x12−2x1x2=0,−x12+4x2=0. 3x_1^2 - 2x_1x_2 = 0, \quad -x_1^2 + 4x_2 = 0.

with a solution at x1=6x_1 = 6, x2=9x_2 = 9.

  • Note that for x1x_1 fixed at x1=6x_1 = 6, the objective attains a relative minimum w.r.t. x2x_2 at $x_2 = 9.
  • Conversely, with x2x_2 fixed at x2=9x_2 = 9, the objective attains a relative minimum w.r.t. x1x_1 at x1=6x_1 = 6.

Despite this fact, the point x1=6x_1 = 6, x2=9x_2 =9 is not a relative minimum because the Hessian matrix is

𝐅(𝐱)=[6x1−2x2−2x1−2x14];𝐅(𝐱*)=[18−12−124]⋡0. \bm{F}(\bm{x}) = \begin{bmatrix} 6x_1 - 2x_2 & -2x_1 \\ -2x_1 & 4 \end{bmatrix}; \qquad \bm{F}(\bm{x}^\ast) = \begin{bmatrix} 18 & -12 \\ -12 & 4 \end{bmatrix} \nsucceq 0.

Second-Order Sufficient Conditions

Proposition (Second-Order Sufficient Conditions)

Let f∈C2f \in C^2 be a function defined on a region in which the point 𝐱*\bm{x}^\ast is an interior point. Suppose

∇f(𝐱*)=0(6) \nabla f(\bm{x}^\ast) = 0 \qquad(6) 𝐅(𝐱*)≻0(7) \bm{F}(\bm{x}^\ast) \succ 0 \qquad(7)

Then 𝐱*\bm{x}^\ast is a strict relative minimum of ff.

Proof

Since 𝐅(𝐱*)\bm{F}(\bm{x}^\ast) is positive definite, there is an a>0a > 0 such that for all 𝐝\bm{d}, 𝐝⊤𝐅(𝐱*)𝐝≥a|𝐝|2\bm{d}^\top \bm{F}(\bm{x}^\ast)\bm{d} \geq a |\bm{d}|^2. By Taylor’s Theorem (with remainder)

f(𝐱*+𝐝)−f(𝐱*)=12𝐝⊤𝐅(𝐱*)𝐝+o(|𝐝|2)≥a2|𝐝|2+o(|𝐝|2). f(\bm{x}^\ast + \bm{d}) - f(\bm{x}^\ast) = \frac{1}{2} \bm{d}^\top \bm{F}(\bm{x}^\ast)\bm{d} + o(|\bm{d}|^2) \geq \frac{a}{2}|\bm{d}|^2 + o(|\bm{d}|^2).

For small |𝐝||\bm{d}|, the first term on the right dominates the second, implying that both sides are positive for small 𝐝\bm{d}.

Properties of Convex Functions

Proposition.

Let f1f_1 and f2f_2 be convex functions on the convex set Ω\Omega. Then the function f1+f2f_1 + f_2 is convex on Ω\Omega.

Proof.

Let 𝐱1,𝐱2∈Ω\bm{x}_1, \bm{x}_2 \in \Omega, and 0<α<10 < \alpha < 1. Then

f1(α𝐱1+(1−α)𝐱2)+f2(α𝐱1+(1−α)𝐱2)≤α[f1(𝐱1)+f2(𝐱2)]+(1−α)[f1(𝐱1)+f2(𝐱2)]. \begin{align} &f_1(\alpha \bm{x}_1 + (1-\alpha)\bm{x}_2) + f_2(\alpha \bm{x}_1 + (1-\alpha)\bm{x}_2) \\ &\leq \alpha [f_1(\bm{x}_1) + f_2(\bm{x}_2)] + (1-\alpha)[f_1(\bm{x}_1) + f_2(\bm{x}_2)]. \end{align}

Proposition.

Let ff be a convex function over the convex set Ω\Omega. Then the function afaf is convex for any a≥0a \geq 0.

Corollary

A conic combination of convex function a1f1+a2f2+⋯+amfma_1f_1 + a_2f_2 + \cdots + a_mf_m is again convex.

Proposition.

Let ff be a convex function on a convex set Ω\Omega. The (sublevel) set Γc={𝐱:𝐱∈Ω,f(𝐱)≤c}\Gamma_c = \{\bm{x}: \bm{x} \in \Omega, f(\bm{x}) \leq c\} is convex for every real number cc.

Proof.

Let 𝐱1,𝐱2∈Γc\bm{x}_1, \bm{x}_2 \in \Gamma_c. Then f(𝐱1)≤cf(\bm{x}_1) \leq c, f(𝐱2)≤cf(\bm{x}_2) \leq c and for 0<α<10 < \alpha < 1,

f(α𝐱1+(1−α)𝐱2)≤αf(𝐱1)+(1−α)f(𝐱2)≤c. f(\alpha \bm{x}_1 + (1-\alpha)\bm{x}_2) \leq \alpha f(\bm{x}_1) + (1-\alpha)f(\bm{x}_2) \leq c.

Thus α𝐱1+(1−α)𝐱2∈Γc\alpha \bm{x}_1 + (1-\alpha)\bm{x}_2 \in \Gamma_c.

Corollary

Since the intersection of convex sets is also convex, the set of points simultaneously satisfying

f1(𝐱)≤c1,f2(𝐱)≤c2,…,fm(𝐱)≤cm, f_1(\bm{x}) \leq c_1, \;\; f_2(\bm{x}) \leq c_2, \ldots, f_m(\bm{x}) \leq c_m,

where each fif_i is a convex function, defines a convex set.
